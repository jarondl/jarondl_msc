Module reference
===============================
Here we document all of the modules (a.k.a ``.py`` files), except
for the ref:`libdl_ref` ones. Notice the 
'source' button on the right, allowing you to see the source code directly.

Two modules are more basic: :py:mod:`jarondl_msc.geometry` and :py:mod:`jarondl_msc.models`. 
By model we mean a realization of some system. :py:mod:`jarondl_msc.pta_chain_g` 
is responsible for data creation and plotting of all ``PTA`` related data.
:py:mod:`jarondl_msc.banded_bloch_ev` is a very small file for dealing with the banded
bloch situation. :py:mod:`jarondl_msc.ptsplot` is for the older ``PTS`` project, 
and it is less organized. Lastly, :py:mod:`jarondl_msc.dtsplot` is a one-time
job for the files we have received from Kottos.
